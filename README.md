# Netflix Landing Page Clone 🎬

A responsive web replica of the Netflix landing page, built entirely from scratch to demonstrate proficiency in modern front-end web development. This project focuses on building a pixel-perfect, mobile-first User Interface (UI) without relying on external CSS frameworks like Bootstrap or Tailwind.

## 🚀 Key Features

* **Responsive Design:** Fluid layout that adapts seamlessly across desktop, tablet, and mobile devices.
* **Modern CSS Layouts:** Heavy utilization of CSS Flexbox and CSS Grid for complex component alignment and structuring.
* **Interactive UI Elements:** Custom hover effects, transitions, and styling to match the premium feel of a real-world streaming platform.
* **Semantic HTML:** Clean, readable, and accessible HTML5 structure.

## 💻 Tech Stack

* **Markup:** HTML5
* **Styling:** CSS3 (Flexbox, Grid, Custom Variables)
